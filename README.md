# SudokuSolver
Simple sudoku solving app for iOS

## Environment
- Xcode 9.4.1

## Setup
- Checkout the project
- No building of packages needed! Simply launch Xcode from the `SudokuSolver.xcodeproj` file

## Tests
- Tests are located under the `SudokuSolverTests` folder
